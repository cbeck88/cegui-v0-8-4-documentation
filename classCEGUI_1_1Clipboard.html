Detailed Description

Defines a clipboard handling class.

Usually, there is just one instance of this class, owned by CEGUI::System, it contains internal CEGUI clipboard that may be (optionally) synchronised with native clipboard if user sets NativeClipboardProvider with:
CEGUI::System::getSingleton()->getClipboard()->setNativeProvider(customProvider)
Where customProvider is of course user implemented clipboard provider.
